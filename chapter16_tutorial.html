16.1 Transverse Waves

A transverse wave has medium displacement perpendicular to the direction of travel (e.g., a wave on a stretched string). A source typically sets frequency f while the medium sets wave speed v; then wavelength is λ = v/f.

v = fλ k = 2π/λ ω = 2πf

Standard traveling-wave form

y(x,t) = ym cos(kx − ωt + φ)

The sign convention indicates travel in +x (use +ωt for travel in −x).

16.2 Wave Speed on a Stretched String

For small transverse oscillations of a string under tension, the wave speed depends on the string tension and its linear mass density.

Wave speed (string)

v = √(T/μ)

T is tension (N), μ is linear density (kg/m).

Quick check: units

T/μ has units (N)/(kg/m) = (kg·m/s²)/(kg/m) = m²/s², so √(T/μ) is m/s.

16.3 Energy and Power of a Wave on a String

A traveling wave transports energy. For sinusoidal waves on a string, average power scales with the square of amplitude and with frequency squared: bigger amplitude or higher frequency means more power.

Key scalings (sinusoidal string wave)

P̄ ∝ μvω2ym2

Exact prefactors depend on the detailed derivation; the proportionality is the most important memory aid.

16.4 The Wave Equation

The wave equation is a differential equation whose solutions include traveling waves. In 1D it ties the curvature of the wave shape to how the shape evolves in time.

1D wave equation

∂2y/∂x2 = (1/v2) ∂2y/∂t2

If a function y(x,t) satisfies this equation, it represents a wave traveling with speed v.

16.5 Interference of Waves

When waves overlap, the net displacement is the sum of displacements (superposition). The resulting amplitude depends on relative phase: in-phase waves reinforce; out-of-phase waves can cancel.

Phase difference from path difference

Δϕ = 2π(Δr/λ)

Always decide what is held constant: same time at a point (standing patterns), or same position for a moving observer, etc.

16.6 Phasors

Phasors represent sinusoidal oscillations as rotating vectors in the complex plane. Adding waves with the same frequency becomes vector addition of their phasors, making interference and phase shifts much easier to compute.

Same-frequency addition

A cos(ωt) + B cos(ωt + δ) = C cos(ωt + φ)

C and φ come from phasor geometry (law of cosines and sines).

16.7 Standing Waves and Resonance

Standing waves arise from two waves traveling in opposite directions (often from reflection at a boundary). Nodes and antinodes form, and only certain wavelengths fit boundary conditions.

String fixed at both ends

λn = 2L/n
fn = nv/(2L)   (n = 1,2,3,…)
Worked example: fundamental frequency of a string

Given: L = 0.65 m, tension T = 120 N, μ = 1.8×10−3 kg/m.

  1. Compute wave speed: v = √(T/μ) = √(120/1.8×10−3) = 258 m/s.
  2. Fundamental frequency: f1 = v/(2L) = 258/(1.30) = 198 Hz.

Answer: The fundamental is about 200 Hz.

Study tip: separate wave kinematics (v = fλ, k, ω) from boundary conditions (what λ values fit). Use phasors to add same-frequency waves quickly.
