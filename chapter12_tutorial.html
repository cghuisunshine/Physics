12.1 Equilibrium

Static equilibrium means an object has no linear acceleration and no angular acceleration. That requires both net force and net torque to be zero.

ΣF = 0 Στ = 0 Choose pivot wisely

Equilibrium conditions

ΣFx = 0, ΣFy = 0, …
Στ = 0 (about any axis)

12.2 Examples of Static Equilibrium

The standard workflow: draw a free-body diagram, pick a pivot to simplify torques, write ΣF = 0 and Στ = 0, then solve.

Worked example pattern: uniform beam + supports
  1. Place the beam, supports, loads, and the weight of the beam at its center of mass.
  2. Take torques about one support to eliminate that unknown force.
  3. Use ΣFy = 0 to find the other support force.

12.3 Elasticity

Elasticity describes how materials deform under applied forces and how they return when forces are removed (within the elastic limit). The response is often linear for small strains.

Core definitions

stress = F/A
strain = ΔL/L

Elastic moduli

Y = (stress)/(strain)   (Young’s modulus)
B = −ΔP/(ΔV/V)   (bulk modulus)
S = (shear stress)/(shear strain)   (shear modulus)

These are material properties; the geometry enters through A, L, and how the load is applied.

Study tip: equilibrium problems are bookkeeping: pick axes, pick a pivot, then write the two sets of equations (forces + torques). Elasticity problems are bookkeeping too: identify stress and strain, then apply the right modulus.
