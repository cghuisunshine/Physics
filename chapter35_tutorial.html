35.1 Light as a Wave

Key ideas

Optical interference happens because light behaves like a wave: at a point, fields add (superposition), and intensity depends on the square of the resulting amplitude. Stable interference requires coherent waves (a stable phase relationship).

Superposition Phase matters Intensity ~ amplitude2

Index and wavelength

n = c/v     λn = λ/n

Frequency f stays the same when light enters a medium; v and λ change.

35.1.1 Huygens’ Principle and Refraction

Huygens’ principle (concept)

Every point on a wavefront acts as a source of secondary wavelets. A later wavefront is the surface tangent to these wavelets. Refraction arises because wave speed differs between media, causing the wavefront to pivot as it crosses the interface.

Snell’s law (as a wave-speed statement)

n1 sinθ1 = n2 sinθ2

35.1.2 Wavelength in a Medium

Internal wavelength

λn = λ/n
Worked Example: Wavelength in glass

Given: λ = 600 nm in vacuum, glass index n = 1.50.

  1. λn = λ/n = 600/1.50 = 400 nm.

Answer: λ in glass is 400 nm (frequency unchanged).

35.1.3 Phase Difference Toolbox

Path and phase

Δφ = (2π/λ)ΔL    (same medium)
Optical path length: OPL = nL
Δφ = (2π/λ)(n1L1 − n2L2)

When comparing rays that travel through different media, use optical path length (nL). A phase flip of π (half a cycle) can also occur on reflection from a higher-index boundary (thin films).

35.2 Young’s Interference Experiment

Key ideas

Two coherent sources (or two slits fed by one source) create an interference pattern on a distant screen. Bright and dark fringes come from constructive and destructive interference at different viewing angles.

35.2.1 Path Difference Conditions

Geometry (small-angle, far field)

ΔL ≈ d sinθ

Bright and dark fringes

Bright: d sinθ = mλ    (m = 0, ±1, ±2, …)
Dark: d sinθ = (m + 1/2)λ
Worked Example: First dark fringe angle

Given: Slit separation d = 0.20 mm, wavelength λ = 500 nm.

  1. First dark: d sinθ = (1/2)λ ⇒ sinθ = λ/(2d).
  2. sinθ = (5.0×10−7)/(2·2.0×10−4) = 1.25×10−3.
  3. θ ≈ 1.25×10−3 rad = 0.0716°.

Answer: θ ≈ 0.072° for the first dark fringe.

35.2.2 Fringe Spacing on a Screen

Small-angle mapping

y ≈ L tanθ ≈ L sinθ

Fringe spacing

Δy = Lλ/d
Worked Example: Fringe spacing

Given: L = 2.0 m, d = 0.30 mm, λ = 600 nm.

  1. Δy = Lλ/d = 2.0(6.0×10−7)/(3.0×10−4).
  2. Δy = 4.0×10−3 m = 4.0 mm.

Answer: Adjacent bright fringes are separated by 4.0 mm.

35.3 Interference and Double-Slit Intensity

Key ideas

Interference describes where fringes land; the intensity formula tells you how bright they are. For two equal slits, intensity varies smoothly between 0 and a maximum, not just “bright or dark”.

Phase form

I = 4I0cos2(Δφ/2)    (equal sources)

Here I0 is the intensity from one slit at the screen point.

35.3.1 Two-Slit Intensity Formula

In terms of path difference

Δφ = (2π/λ)ΔL
I(θ) = 4I0cos2(πΔL/λ) = 4I0cos2(πd sinθ/λ)
Worked Example: Intensity at a given path difference

Given: ΔL = λ/3 and the two slits contribute equal intensity I0 each.

  1. I = 4I0cos2(πΔL/λ) = 4I0cos2(π/3).
  2. cos(π/3) = 1/2 ⇒ I = 4I0(1/4) = I0.

Answer: I equals the intensity from one slit alone.

35.3.2 Missing Orders and Envelopes

When slits have finite width

Real slits have width a, so single-slit diffraction creates an envelope that modulates the double-slit interference pattern. Some interference maxima can “go missing” when a bright fringe would fall at a diffraction minimum.

Useful idea

Diffraction minima: a sinθ = mλ    (m = ±1, ±2, …)

35.4 Interference from Thin Films

Key ideas

In a thin film, two reflected rays can interfere: one reflects from the top surface, another from the bottom surface after traveling through the film. Whether you get bright or dark reflection depends on path difference in the film and possible phase shifts upon reflection.

35.4.1 Phase Shifts on Reflection

Rule of thumb

Reflection Phase shift Memory cue
Low n → high n π (half-cycle) “Hard boundary” flip
High n → low n 0 No flip

35.4.2 Film Thickness Conditions (Normal Incidence)

Extra path in film

ΔLfilm = 2nt

Combine the film path difference with any reflection phase flips (0 or π) to decide constructive vs destructive interference for the reflected light.

Common case: one π flip

Bright reflection: 2nt = (m + 1/2)λ
Dark reflection: 2nt = mλ
Worked Example: Minimum thickness for bright reflection

Given: Soap film (n = 1.33) in air, λ = 550 nm, normal incidence, assume one phase flip.

  1. First bright (m = 0): 2nt = (1/2)λ ⇒ t = λ/(4n).
  2. t = 550/(4·1.33) nm = 103 nm.

Answer: t ≈ 1.0×102 nm.

35.5 Michelson’s Interferometer

Key ideas

A beam splitter sends light along two paths that recombine. If one mirror moves, the optical path difference changes and the output intensity cycles through bright and dark fringes. Counting fringes measures tiny displacements and wavelength.

35.5.1 Fringe Counting for Mirror Displacement

Path difference change

Δ(OPL) = 2Δx    (mirror moves by Δx)

Fringe count

Number of fringes N = 2Δx/λ
Worked Example: Displacement from fringe count

Given: λ = 633 nm (He–Ne laser). You count N = 200 fringes as a mirror moves.

  1. Δx = Nλ/2 = 200(633×10−9)/2 = 6.33×10−5 m.
  2. Δx = 63 μm.

Answer: Mirror displacement is about 6.3×10−5 m.

Study tip: for interference problems, write everything in terms of phase. Use Δφ = (2π/λ)Δ(OPL), then apply “bright ↔ Δφ = 2mπ” and “dark ↔ Δφ = (2m + 1)π”. This keeps path differences, media (n), and reflection flips in one place.
