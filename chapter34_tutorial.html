34.1 Images and Plane Mirrors

Key ideas

An image is a reproduction of an object via light rays. If rays actually converge at the image location, the image can be formed on a screen and is real. If the eye/brain must “trace back” diverging rays to an apparent source, the image is virtual.

Real: can project Virtual: apparent source Ray diagrams decide

Plane mirror result

i = −p

Object distance p > 0; plane-mirror image distance i < 0 (virtual, behind mirror).

34.1.1 Real vs Virtual

Quick test

If you can put a screen at the image location and see a focused picture, the image is real. If the “image” disappears on a screen and only exists when you look into the system, it is virtual.

34.1.2 Plane Mirror Distance

Image position and size

A plane mirror produces an upright virtual image the same size as the object, located the same distance behind the mirror as the object is in front. The mirror reverses left–right (lateral inversion).

m = hi/ho = +1

34.2 Spherical Mirrors

Key ideas

Spherical mirrors approximate focusing for rays close to the axis (paraxial rays). A concave mirror can form real images; a convex mirror forms only virtual images.

Concave: f > 0 Convex: f < 0 Paraxial approximation

Focal length

f = R/2

R is mirror radius of curvature (sign follows mirror type).

34.2.1 Mirror Equation

Core relation

1/p + 1/i = 1/f

Sign convention used here: p > 0 for a real object in front of the mirror. Image distance i is positive for a real image in front of the mirror and negative for a virtual image behind it. f is positive for concave mirrors and negative for convex mirrors.

34.2.2 Magnification and Sign

Magnification

m = hi/ho = −i/p

m > 0 means the image is upright; m < 0 means inverted. |m| gives the size ratio. Combine the sign of i with the sign rule above to interpret the result.

Worked Example: Concave mirror image location

Given: Concave mirror f = +10 cm, object distance p = 30 cm.

  1. Mirror equation: 1/i = 1/f − 1/p = 1/10 − 1/30 = (3 − 1)/30 = 2/30.
  2. i = 15 cm (positive → real image in front of mirror).
  3. m = −i/p = −15/30 = −0.50 (inverted, half-size).

Answer: i = +15 cm, m = −0.50.

34.3 Spherical Refracting Surfaces

Key ideas

Refraction at a curved interface (like air–glass with a spherical boundary) can form images even without a full lens. The image location depends on the refractive indices and the curvature.

34.3.1 Refraction at a Curved Interface

Gaussian (paraxial) refraction formula

n1/p + n2/i = (n2 − n1)/R

Here n1 is the index on the object side and n2 on the image side. R is the radius of curvature of the interface (its sign depends on which side the center of curvature lies on). This relation is a building block for lens design.

34.4 Thin Lenses

Key ideas

A thin lens is modeled as bending rays at a single plane. Converging lenses (positive f) can form real images on a screen; diverging lenses (negative f) form virtual images.

Converging: f > 0 Diverging: f < 0 Same equation as mirrors

Thin-lens equation

1/p + 1/i = 1/f

Same algebraic form as the mirror equation; only the sign rules differ by system.

34.4.1 Thin-Lens Equation

Sign convention (common)

Quantity Positive Negative
p (object distance) Real object (on incoming-light side) Virtual object (rare in basic problems)
i (image distance) Real image (on outgoing-light side) Virtual image (same side as object)
f (focal length) Converging lens Diverging lens

34.4.2 Lens Magnification

Magnification

m = hi/ho = −i/p
Worked Example: Converging lens image and magnification

Given: Converging lens f = +15 cm, object distance p = 30 cm.

  1. 1/i = 1/f − 1/p = 1/15 − 1/30 = 1/30 ⇒ i = 30 cm (real image).
  2. m = −i/p = −30/30 = −1.0 (inverted, same size).

Answer: i = +30 cm, m = −1.0.

Worked Example: Diverging lens (virtual image)

Given: Diverging lens f = −20 cm, object distance p = 40 cm.

  1. 1/i = 1/f − 1/p = (−1/20) − (1/40) = −3/40.
  2. i = −13.3 cm (negative → virtual image on object side).
  3. m = −i/p = −(−13.3)/40 = +0.333 (upright, reduced).

Answer: i ≈ −13.3 cm, m ≈ +0.33.

34.5 Optical Instruments

Key ideas

Optical instruments arrange lenses (and sometimes mirrors) to control where the final image forms and to increase the angular size of an object at the eye. The key design idea is often: create a convenient intermediate image, then use a second element to view it.

34.5.1 Magnifier Idea

Simple magnifier

A converging lens used as a magnifier places the object closer than the focal length so the image is virtual, upright, and enlarged. Your eye then views a larger angular size than it would without the lens.

34.5.2 Microscope and Telescope Idea

Two-stage imaging

A microscope typically uses an objective lens to form a real, magnified intermediate image and an eyepiece to magnify that image as a virtual image for comfortable viewing. A telescope uses a similar “objective then eyepiece” pattern, but the objective forms an image of a distant object near its focal plane.

34.6 Three Proofs

Key idea

The imaging equations (mirror, lens, spherical refraction) can be derived geometrically using ray diagrams and similar triangles. The derivations rely on “small-angle” approximations so that rays close to the axis behave predictably.

34.6.1 Paraxial Approximation

Small-angle simplifications

For small angles (in radians), sinθ ≈ tanθ ≈ θ. This lets geometry turn ray slopes into simple ratios of heights and distances, producing equations like 1/p + 1/i = 1/f. Away from the axis, real optics show aberrations (blurring).

Study tip: always decide real vs virtual first (sign of i), then interpret the magnification sign (upright vs inverted). If you get an “impossible” result, sketch 2–3 principal rays to sanity-check your sign convention.
